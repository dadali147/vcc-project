## 使用方式

### 快速启动
```bash
cd ~/workspace/vcc-project/vcc-web
npm install  # 如未安装依赖
npm run dev  # 启动开发服务器
# 访问 http://localhost:5174
```

### 在代码中使用
```vue
<!-- 模板中 -->
<h1>{{ $t('dashboard.welcome') }}</h1>

<!-- 脚本中 -->
<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>
```

### 切换语言
```javascript
import { setLocale } from '@/locales'
setLocale('en-US')  // 切换到英文
```
