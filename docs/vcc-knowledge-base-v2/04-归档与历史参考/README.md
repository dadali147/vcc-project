# 04-归档与历史参考

本目录用于承接 VCC 知识库重构过程中的历史文档、旧版参考、草案和过程稿。

## 目录定位

这里不是当前正式知识库，不承担现行规则定义职责。

本目录只用于：
- 保存旧版参考
- 保存草案、过程稿、中间稿
- 保存已被新基线替代但仍需追溯的历史文档
- 为新旧知识库切换提供迁移缓冲和历史上下文

## 使用边界

本目录文档：
- **可以**用于历史追溯、差异比对、背景理解、迁移参考
- **不得**作为开发、派单、联调、测试、验收的主依据
- 与新知识库正式基线冲突时，**一律以新基线为准**

## 归档候选判断

通常应进入归档区的文档类型包括：
- 已被新知识库正式正文替代的旧文档
- 标记为草稿、讨论版、临时版、过程版的文档
- 历史版本的接口、数据库、流程、状态、页面说明
- 迁移过程中形成的映射稿、备注稿、比对稿
- 已废弃方案、已失效规则说明、旧执行口径文档

## 禁止事项

- 禁止把本目录文档当作当前正式规范
- 禁止在本目录持续迭代现行业务规则
- 禁止在新基线未落稳时，用归档文档临时顶替正式文档
- 禁止把归档区当成无分类杂物堆
