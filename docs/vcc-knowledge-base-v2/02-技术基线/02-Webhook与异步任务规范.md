# 02-Webhook与异步任务规范

## 1. 文档定位
本文件统一 Webhook 事件、异步任务状态机、补偿、幂等、超时与数据库承接规则。

## 2. Webhook 规范
- 回调事件进入系统的唯一规则
- 验签、安全、应答、幂等处理
- 失败重试与人工介入边界

## 3. 异步任务规范
- 统一任务状态机
- 轮询与回调协同
- 重试、补偿、超时、人工介入

## 4. 与数据库承接关系
异步任务模型的数据库正式承接见：
- `03-数据库DDL正式基线.md`

正式落库的核心表包括：
- `vcc_async_task`
- `vcc_async_task_log`
- `vcc_async_task_retry`
- `vcc_async_task_compensation`

## 5. 使用原则
- webhook 事件入口与异步任务收敛分层处理
- 规范层描述与数据库落库若有冲突，以正式 DDL 基线为准
- 后续 SQL / 实体映射 / 调度实现统一按正式 DDL 基线落地
